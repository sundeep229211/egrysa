# Support

The open-source alpha is community-supported through GitHub issues. Support is best-effort; there is
no uptime, response-time, compatibility, or security-fix SLA.

Use issues for reproducible bugs, design questions, and deployment patterns with synthetic data.
GitHub Discussions is not enabled. Use GitHub private vulnerability reporting for security issues.
Do not place customer prompts, responses, credentials, confidential taxonomies, or production
configuration in any public channel.

Commercial support is not yet offered. Any future offering will be documented separately and will
not require prompt access by the project maintainers.
