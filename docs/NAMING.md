# Naming screen

Screen date: 2026-07-15

This is a preliminary naming knockout, not trademark clearance or legal advice. It is intended to
avoid obvious conflicts before counsel performs jurisdiction-, class-, phonetic-, and
translation-aware searches.

## Selection

**Egrysa** is the final product and repository name selected on 2026-07-15. The founder reported
that external legal screening was complete before authorizing the rename. This repository does not
reproduce counsel's work or make an independent trademark-clearance claim.

**Ciphrel** was the leading candidate and **Egrysa** the fallback before the final selection. The
historical screen below is retained as preliminary context only. Domain status, developer
namespaces, and common-law use can change and must be checked again immediately before publication.

## Screen performed

- Exact-name web search with AI, software, and security context.
- Exact npm and PyPI package-name lookup.
- Verisign `.com` RDAP lookup as a commercial-availability signal only.
- Manual review for pronunciation, descriptiveness, category confusion, and misleading security
  promises.

## Required clearance before selection

Counsel should search exact, phonetic, stem, spacing, translation, and common-law uses in every
launch jurisdiction. The likely software and SaaS classes require particular attention, but counsel
must determine the final goods/services description and classes. Also check company registries,
GitHub organizations, container registries, social handles, and app stores immediately before
filing.

The final publication gate requires:

1. founder confirmation that legal screening is complete;
2. founder approval of pronunciation and positioning;
3. one atomic repository/package rename with signed-release continuity; and
4. a fresh namespace and collision check immediately before publication. Any domain purchase or
   external account acquisition requires separate approval.
